Aviation Analytics Dashboard
Project Overview

Developed an interactive Aviation Analytics Dashboard using Power BI to analyze airline operations, airport performance, passenger satisfaction, weather conditions, and airline route intelligence. The dashboard integrates multiple datasets and provides KPI-driven insights to support operational and strategic decision-making.

Tools & Technologies:
1. Power BI
2. Power Query
3. DAX
4. Data Modeling (Star Schema)
5. Excel / CSV Datasets

Key Features:
1. Flight Operations Analysis
2. Airport Performance Monitoring
3. Passenger Satisfaction Analytics
4. Weather Impact Assessment
5. Route Intelligence & Connectivity Analysis
6. Interactive KPI Dashboards
7. Multi-page Navigation System

Datasets Used:
1. Airlines Dataset (Fact Table)
2. Airports Dataset
3. Airlines Dataset
4. Passenger Satisfaction Dataset
5. Global Weather Repository
6. Routes Dataset

Dashboard Pages:
Executive Overview:
1. Total Flights
2. Delayed Flights
3. Passenger Satisfaction
4. Flight Trends

Airport Traffic Analysis:
1. Airport Analytics
2. Airport Performance
3. Traffic Distribution
4. Delay Analysis
5. Country-wise Airport Insights

Passenger Insights:
1. Customer Segmentation
2. Satisfaction Analysis
3. Service Quality Evaluation
4. Delay Impact Assessment

Weather & Route Analytics:
1. Weather Distribution
2. Route Intelligence
3. Environmental Analysis
4. Airline Connectivity

Key Business Insights:
1. Analyzed 98K+ flight records and 104K+ passenger records.
2. Identified major airport hubs and high-traffic routes.
3. Evaluated passenger satisfaction across travel classes.
4. Analyzed the impact of weather conditions on airline operations.
5. Generated actionable business insights through interactive visualizations.
